# AGENTS.md Examples

This directory contains example AGENTS.md files for different project types.

## What is AGENTS.md?

AGENTS.md is a file that tells opsy about your project - its structure, conventions, and how to work with it. Place it in your project root and commit to version control.

## Available Examples

- [terraform-project.md](./terraform-project.md) - For Terraform infrastructure projects
- [kubernetes-app.md](./kubernetes-app.md) - For applications deployed to Kubernetes
- [monorepo.md](./monorepo.md) - For monorepo with multiple services

## Usage

1. Copy the relevant example to your project root
2. Rename to `AGENTS.md`
3. Customize for your project
4. Commit to version control

Or run `/init` in opsy to auto-generate one.
